# My Portfolio Website

## Description

This repository contains the code for my personal portfolio website which includes the notable experiences and achievements throughout my career.

## Approach

I chose to complete this project to practise AI prompting strategies. I had spoken with former colleagues about different approaches to prompting AI models and ideas for how to leverage them effectively. For this particular project, I wanted to start off with a relatively simple prompt [mention initial prompt file name here] and let AI devise a plan for the full implementation, creating more thorough documentation for itself to reference. I also had the model keep documentation up-to-date as the application evolved and track the progress to keep things organized.

Another intention behind this project was to become familiar with configuring and hosting a website in AWS. I completed the Certified Cloud Practitioner certification but have not had the chance to apply what I've learned for any personal or professional work.
