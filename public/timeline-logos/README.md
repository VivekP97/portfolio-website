# Timeline logos

Place PNG logo files here for work experience, education, and certification timeline items.

Reference them in `resume-data/timeline.json` with the optional `logo` field (filename only, e.g. `"logo": "company.png"`).

- **Work experience** – use `logo` to show a company/institution logo instead of the default briefcase icon.
- **Education** – use `logo` to show an institution logo instead of the default graduation cap icon.
- **Certification** – use `logo` to show an issuer logo instead of the default award icon.

If `logo` is omitted for an item, the default icon for that type is used.
